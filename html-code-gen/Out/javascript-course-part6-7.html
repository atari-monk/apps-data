  • Learning coding

    Have a clear goal.
    Understand and type code from tutorials.
    Learn and reinforce with notes and exercises (calculate something personal, codewars page).
    Code your own projects.
    Dont obsess about clean code or that you will never know everything.
    Dont learn in isolation. Some community is beneficial.
    Only fixing problems and never stoping it, will make you competent. Learning never stops.

  • Solving problems

    1. 100% understand clear picture of the problem.
    2. Divide and conquer: break big into smaller sub-problems.
    3. Research as much as you have to.
    4. For bigger problems, write pseudo-code before actual code.
