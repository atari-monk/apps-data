• Part 12; this keyword.

    window - global JS object.
    Created for every function execution context.
    Points to 'owner' of function.
    Method (atached to object) has this pointing to calling it object.
    Function has this undefined in strict mode.
    Arrow Function has this of parent/surrounding scope function.
    Event listener has this pointing to DOM element.
    console.log(this) proves that global this points to global window object.
    Function is a variable so it can be assigned to variables.
    It can be 'borowed' betwween objects or assigned from object to some global variable. Depending on from what context it's called, this will change accordingly.

  • Function vs Arrow Function in object

    Arrow Function as method have this pointing to global window, not object.
    Never use it as method.
    To call function in method (object's function) use Arrow Function becouse then this will point to object as expected (when method or method expression will get undefined this).

  • Primitives vs Reference Types

    Primitives are stored in the callstack (execution contexts).
    Objects are stored in the heap.
    Primitives has Identifiers pointing to address and value on callstack.
    Identifiers can be assign to any address.
    Objects have identifiers pointing to address and value on callstack, that value is address on the heap, pointing to object value.
    to copy object: const objCopy = Object.assign({}, obj); (shallow copy - dosent copy internal objects)

  • Part 13. Modern syntax. Destructuring Arrays. Gets variables from array.

    const [x,y,z] = arr; is same as const a = arr[0];...;c = arr[2]
    const [x, ,z] = arr; to skip values.
    let [x, ,z] = arr; [x,z] = [z,x]; to switch values in variables.
    const [x, y] = f(); to get multiple values from function returning array.
    const nested = [2, 4, [5, 6]]; const [i, , [j, k]] = nested;
    const [p = 1, q = 1, r = 1] = [8, 9]; this gets 8, 9, 1 (default values)

  • Destructuring objects. Gets variables from object.

    const { name, openningHours, categories } = obj; this takes values form object properties.
    const { menu = [], starterMenu:starters = [] } = obj; change variable names, give default value in case no value.
    const obj = { a: 23, b: 7, c: 12 }; ({a, b}) = obj; mutate variables when destructuring object.
    const { fri: { open: o, close: c}} = openningHours; nested destructuring object with new variable names.
    Destructuring object can be done as a parameter in function or method.

  • Array Spread operator

    Useable on Iterables(not objects): arrays, strings, maps, sets and objects.
    const newArr = [ 1, 2, ...arr]; expands arr elements to newArr.
    console.log(...log); logs individual elements of array expanded into method.
    const newMenu = [...restaurant.mainMenu, 'Gnocci']; spread when creating new array.
    const arrCopy = [...arr]; simpler syntax for array copy.
    const menu = [...obj.menuA, ...obj.menuB]; joining arrays.
    const str = 'Jonas'; const letters = [...str, '', 'S.'];
    const objCopy = {...obj}; simpler syntax for object copy.

  • Rest operator

    const arr = [1,2, ...[3, 4]]; spread - on right of =
    const [a, b, ...others] = [1, 2, 3, 4, 5]; destructuring with rest - on left of =
    const add = function (...numbers); aggregates any number of separete parameters to array numbers, also can be called with spread array.
    const add = function (x, y, ...otherNumbers); gets 'unused' params to array in function.

  • Logic operators expanded

    const x = obj.num || 10; 10 if undefined/0 (when falsey). Short circuts on first truethy.
    obj.method && obj.method(x,y). Call method only if it exists. Short circuts on first falsey.

  • Null coalescing operator

    const x = obj.num ?? 10; 10 when nullish: null, undefined.
    ||=, &&=, ??= are also in JS.
