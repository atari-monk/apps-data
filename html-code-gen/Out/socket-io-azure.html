• Setup project. In git bash or VSCode terminal.

    cd chat-server
    npm init
    npm install socket.io
    npm install express

  • To .gitignore add (if not there)

    node_modules
    .vscode

  • Add index.js with server code.

  • Run it locally. Ctrl+C to kill it.

    node index.js

  • Add folder chat-client.

  • Add index.html with chat client code.

  • With live server extension, open two instances of chat client page.

  • Open git bash in chat-server folder and run server.

    node index.js

  • Send message with one client. Check if it was recive by server and send to another clinet.

  • Deploy to Azure.

  • VSCode Azure extension. App Services. Create New Web App.

    details

    example name

    atari-monk-chat-server

  • Right click on chat-server, Deploy to Web App... .

  • Choose web app name and assign folder to it in popup.

  • After server is deployed. Deploy client page with github pages.

  • Test it.
