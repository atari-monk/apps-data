• Multi Cursor.

    Select name.
    Ctrl + D, multiple times.
    Gives multi cursor on occurences of name below selected.

  • Workspace.

    Combines multiple solutions in one instance of VSCode.
    Open folder of first solution.
    File->Add Folder To Workspace. As many as you need.
    File->Save Workspace As.... I save in .vscode of solution that workspace is created for.
    File->Close Workspace on finish.
    File->Open Workspace from File... to continue.
