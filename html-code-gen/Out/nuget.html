• Create new github repo

  • Open folder with VSCode

  • Setup project.

    dotnet new sln
    dotnet new classlib -n Connection.Builder
    dotnet sln add ./Connection.Builder/

  • In csproj file set properties describing nuget

    details

    Id name, unique in nuget.org

    version

    author

    company name

    tags important for searching nuget

    description

    Connection.Builder.15.11.2022.001.Test
    1.0.0
    atari-monk

    tag1;tag2
    Builds ConnectionString.

  • Build

    dotnet build

  • Use nuget command.

    details

    nuget path in output

    dotnet pack
