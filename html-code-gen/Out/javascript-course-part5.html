    /*Read:*/ jonas.lastName, jonas['lastName'], jonas[expression], jonas.friends[0], jonas.friends.length
    /*Add:*/ jonas.lastName = 'x', jonas['lastName'] = 'y', jonas[expression] = 'z'

  • Objects Methods

    function can be added to object, calcAge: function() {}
    Access: jonas.calcAge(1991); jonas.['calcAge'](1991);
    Use this. to access object instance in object's methods.
    Use this.x = expression; to create new objects property in object's methods.
    This way you can invoke method once and then get value from created property next time.
