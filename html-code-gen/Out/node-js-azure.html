• Create new github repo, open git bash in Code folder, clone it.

    git clone https://github.com/atari-monk/web-dev-basics.git

  • Verify that Node.js is installed.

    node --version

  • Create a Node.js application using the Express Generator.

    details

    repo folder

    app name

    cd web-dev-basics && npx express-generator myExpressApp --view ejs

  • Install the NPM packages to the app.

    details

    app name

    cd myExpressApp && npm install

  • Start the development server with debug information.

    details

    app name

    DEBUG=myExpressApp:* npm start

  • Test in the browser.

    http://localhost:3000
